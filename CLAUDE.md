## Project Overview

UIGen is an AI-powered React component generator with live preview. Users describe components in a chat interface, Claude generates the code using tools, and a live preview renders the result in-browser via a virtual file system.

## Commands

- `npm run setup` — Install deps, generate Prisma client, run migrations
- `npm run dev` — Start dev server with Turbopack (port 3000)
- `npm run build` — Production build
- `npm run lint` — ESLint
- `npm run test` — Vitest (all tests)
- `npx vitest run src/path/to/test.ts` — Run a single test file
- `npx prisma generate` — Regenerate Prisma client after schema changes
- `npx prisma migrate dev` — Create/apply migrations
- `npm run db:reset` — Reset database (destructive)

## Architecture

### Core Flow

1. **Chat** (`src/app/api/chat/route.ts`): POST endpoint receives messages + serialized virtual file system. Uses Vercel AI SDK `streamText` with two tools (`str_replace_editor`, `file_manager`) that operate on a `VirtualFileSystem` instance server-side.

2. **Virtual File System** (`src/lib/file-system.ts`): In-memory tree structure (no disk writes). Serialized as JSON between client and server. The client-side context (`src/lib/contexts/file-system-context.tsx`) mirrors tool calls to keep the UI in sync.

3. **Live Preview** (`src/components/preview/PreviewFrame.tsx` + `src/lib/transform/jsx-transformer.ts`): Transforms JSX/TSX files using `@babel/standalone`, creates blob URLs, builds an ES module import map, and renders in a sandboxed iframe. Third-party packages are resolved via `esm.sh`. Entry point is `/App.jsx`.

4. **Mock Provider** (`src/lib/provider.ts`): When `ANTHROPIC_API_KEY` is not set, a `MockLanguageModel` returns static component code so the app is functional without an API key.

### Key Patterns

- **Two React Contexts**: `FileSystemProvider` wraps `ChatProvider`. Chat context uses `useChat` from `@ai-sdk/react` and forwards tool calls to the file system context via `handleToolCall`.
- **AI Tools**: `str_replace_editor` (create/str_replace/insert commands) and `file_manager` (rename/delete). Both operate on the `VirtualFileSystem` instance.
- **Auth**: JWT sessions via `jose`, stored in httpOnly cookies. Middleware protects `/api/projects` and `/api/filesystem` routes. Anonymous usage is supported.
- **Path alias**: `@/*` maps to `./src/*` (tsconfig paths).
- **UI Components**: shadcn/ui (new-york style) in `src/components/ui/`.

### Data Model

SQLite via Prisma. Schema is defined in `prisma/schema.prisma` — always reference it to understand the database structure. Two models: `User` (email/password) and `Project` (name, messages as JSON string, data as JSON string of serialized VFS).

### Tech Stack

- Next.js 15 (App Router), React 19, TypeScript
- Tailwind CSS v4, shadcn/ui
- Prisma + SQLite
- Vercel AI SDK + Anthropic Claude (claude-haiku-4-5)
- Vitest + Testing Library + jsdom
